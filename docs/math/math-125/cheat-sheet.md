## Antiderivative Rules

$$\int 0 dx = C$$

$$\int k dx = kx + C$$

$$\int x^n dx = \frac{x^n+1}{n+1}$$

---

## Trigonometric Rules

$$\sin^2 x + \cos^2 x = 1$$

$$\tan^2 x + 1 = \sec^2 x$$

$$\sin(2x) = 2\sin x \cos x$$

$$\cos(2x) = 1 - 2\sin^2 x$$
